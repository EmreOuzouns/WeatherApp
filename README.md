## ✨ Features

- **Physics-Based Canvas Graphics**: Ditch traditional CSS limitations. Features real-time rain fluid dynamics, trigonometric snow fluttering, moving fog mapping, and shooting wind vectors.
- **Glassmorphism Aesthetic**: Apple-inspired translucent layers over real-time reactive backgrounds.
- **Day/Night Cycle Tracking**: Auto-adjusts interface style, star clusters, and moonlight based on the requested location's astronomical sunset & sunrise times natively via Open-Meteo.
- **Real-time Metrics Tracker**: Showcases atmospheric pressure, wind velocities, and specific fishing viability indices based on complex local patterns. 
- **PWA & Offline Native Capabilities**: Installable as a native app on iOS & Android using Service Worker cache optimizations.

## 🚀 Usage

1. **Access the Web App:** Open the application via the live Netlify demo link.
2. **Location Access:** Allow browser location access for instant local weather data, or use the search functionality to look up specific global cities.
3. **PWA Installation:** For the best experience, install the app directly to your device's home screen via your browser settings for a native, full-screen mobile experience.

## 🛠 Tech Stack

- **HTML5 Canvas & Antigravity:** Custom built 2D Particle Engine for Physics Rendering and smooth fluid animations.
- **CSS:** Complex Custom Properties, Backdrop filters for glassmorphism.
- **Vanilla JS:** Zero dependencies, high-perf DOM manipulation.
- **Open-Meteo API:** No-key raw weather dataset integrations.

## 📥 Installation

Simply open `index.html` in any modern web browser or run via a local web server (e.g., `python -m http.server`). No complex build steps or dependencies required.
